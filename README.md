# DEPI-Project-
Skin Cancer

## Getting Started

### Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/SamaHassa/Skin_Cancer_DEPI.git
    ```
2. Navigate to the project directory:
    ```bash
    cd Skin_Cancer_DEPI
    ```
